# 🦷 LumiSmile Website

---

A web-based dental clinic website project focused on creating a clean, modern, and user-friendly interface.
This project was built as part of my learning journey in React, where I developed the application while following and adapting a tutorial for educational purposes.

The website emphasizes strong visual hierarchy, responsive design, and an improved user experience to make information easy to access and visually appealing.

## 🚀 Tech Stack

* React.js
* Tailwind CSS
* JavaScript

## 🎯 Purpose

This project was created for learning and practice, especially to improve my skills in building modern UI with React and Tailwind CSS while understanding real-world layout and design structure.
